# C++ Reference Notes Plan

## Objective
Create a structured knowledge base of C++ language features, standard library (STL), and modern C++ (C++11/14/17/20/23) best practices.

## Key Areas to Document
1. **Language Core**:
   - Types, Objects, and Scope.
   - Functions & Lambda Expressions.
   - Classes & OOP.
   - Templates & Metaprogramming.
   - Concepts (C++20).
2. **Standard Library (STL)**:
   - Containers (vector, list, map, etc.).
   - Algorithms (sort, find, transform, etc.).
   - Iterators.
   - Strings & Input/Output.
3. **Concurrency**:
   - Threads, Mutexes, Futures.
   - Atomic Operations.
   - Memory Model.
4. **Modern C++ Features**:
   - Move Semantics & Smart Pointers.
   - Constexpr & Consteval.
   - Ranges (C++20).
   - Modules (C++20).

- [x] Phase 1: Setup reference links and overview of C++ Standards.
- [x] Phase 2: Core Language features deep dive.
- [ ] Phase 3: STL Container and Algorithm analysis.
- [ ] Phase 4: Advanced topics (Template Metaprogramming, Concurrency).
